# House Price Prediction — Regression Analysis

## Overview
This project predicts house prices based on area-level housing and demographic features using regression models. Unlike the other projects in this portfolio (which were classification problems), this project focuses on **regression** — predicting a continuous numeric value (price) rather than a category.

## Dataset
**USA Housing Dataset** — a clean, beginner-friendly dataset containing area-level housing statistics and the corresponding house price.

Source: [Kaggle — USA Housing](https://www.kaggle.com/datasets/vedavyasv/usa-housing)

| Column | Description |
|---|---|
| Avg. Area Income | Average income of residents in the area |
| Avg. Area House Age | Average age of houses in the area |
| Avg. Area Number of Rooms | Average number of rooms in houses in the area |
| Avg. Area Number of Bedrooms | Average number of bedrooms in houses in the area |
| Area Population | Population of the area |
| Price | Target variable — house price |
| Address | Dropped during cleaning (not predictive) |

## Project Workflow
1. **Data Cleaning** — Dropped the non-predictive `Address` column. Confirmed no missing values (dataset is already clean).
2. **Exploratory Data Analysis (EDA)** — Visualized price distribution, correlation heatmap, scatter plots of each feature against price, and a full pairplot.
3. **Feature Scaling** — Standardized features using `StandardScaler` for the Linear Regression model.
4. **Modeling** — Trained and compared two regression models:
   - Linear Regression
   - Random Forest Regressor
5. **Evaluation** — Compared models using MAE (Mean Absolute Error), RMSE (Root Mean Squared Error), and R² Score.
6. **Feature Importance** — Examined Linear Regression coefficients and Random Forest feature importances to understand which factors drive price the most.

## Key Findings
- **Avg. Area Income** is the strongest driver of house price — higher area income strongly correlates with higher house prices.
- **Avg. Area Number of Rooms** also has a meaningful positive relationship with price.
- **House Age** and **Number of Bedrooms** have a comparatively weaker influence on price in this dataset.
- The relationship between features and price is largely linear, which is reflected in the model performance below.
