# HR Attrition Prediction Using Machine Learning

This repository features a Data Science project focused on analyzing employee data to predict attrition. The goal is to help HR teams identify factors contributing to employee turnover and implement proactive retention strategies.

##  Project Overview
Employee attrition impacts organization stability and cost. This project uses Python to perform comprehensive analysis, handle data preprocessing, and train machine learning models to forecast which employees are most likely to leave.

##  Key Features
* **Exploratory Data Analysis (EDA):** Identifying correlation between department, salary, work-life balance, and attrition rates.
* **Feature Engineering:** Encoding categorical variables and scaling numerical fields for model readiness.
* **Predictive Modeling:** Applying classification algorithms (e.g., Logistic Regression, Decision Trees, or Random Forests) to accurately classify turnover risk.

##  Tech Stack & Tools
* **Language:** Python
* **Libraries:** Pandas, NumPy, Scikit-Learn, Matplotlib, Seaborn
